# Bingo Game and Endless Bingo Game with Score Multipliers

This project has two different apps: Bingo Game and Endless Bingo Game, both
with score multipliers.

## Features

### Bingo Game

- Time limit of 1, 2, 3, or 5 minutes
- Score multiplier based on how quickly each cell is marked
- Bingo score multiplier increases with each bingo
- Endless bingo game with score multipliers

### Endless Bingo Game

- Time limit of 1, 2, 3, or 5 minutes
- Score multiplier based on how quickly each cell is marked
- Bingo score multiplier increases with each bingo
